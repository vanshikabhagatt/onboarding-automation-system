# Onboarding Workflow Automation System

This project automates employee onboarding using Google Workspace tools and Slack integration.

 Features
- Automated email sent on form submission
- Real-time Slack notifications using webhook APIs
- Event-driven workflow using Google Apps Script

 Tech Stack
- Google Forms & Sheets
- Google Apps Script
- Slack Webhooks (API)

 How it Works
1. User fills form
2. Data goes to Google Sheets
3. Script triggers automatically
4. Email + Slack message sent
